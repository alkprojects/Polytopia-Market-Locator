# Polytopia Market Locator

A planning tool for [The Battle of Polytopia](https://polytopia.io) that finds the optimal placement of Markets and supplier buildings (Sawmills, Windmills, Forges, Customs Houses) on your map to maximize stars-per-turn.

Upload a screenshot of your map, align the grid, paint terrain, and the optimizer searches for the highest-value market layout given your available techs and buildings.

## Features

- Screenshot-based map alignment with two-anchor grid scaling
- Full terrain palette (cities, forests, crops, mountains, ore, fish, fruit, etc.)
- Territory assignment per city
- Tech / building toggles (Trade, Construction, Forestry, Mining, Sailing, Chivalry, etc.)
- Building locks for user-placed structures the optimizer must respect
- Live stars/turn readout

## Usage

Open `index.html` in a browser, or visit the live version. A built-in step-by-step guide is in the header.
